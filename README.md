# rotating-images-gallery
Interactive rotating image gallery built with HTML, CSS, and JavaScript.
A modern and interactive image gallery built using HTML, CSS, and JavaScript featuring smooth rotating image effects and dynamic user interaction.

## Features

* Rotating image gallery animation
* Smooth transitions and effects
* Interactive user experience
* Responsive design
* Lightweight and beginner-friendly project
* Pure HTML, CSS, and JavaScript implementation

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

rotating-images-gallery/
│
├── index.html

## Project Goal

This project was created to practice JavaScript DOM manipulation, CSS animations, and interactive UI development while building a visually appealing rotating image gallery.

## Functionality

* Images rotate automatically or through user interaction
* Smooth animation effects
* Dynamic image display
* Responsive layout for different screen sizes

Optimized for:
